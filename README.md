# koa-hello

> clone this as a cli to start your own koa app

## init

```bash
yarn install
```

## development

```bash
yarn run dev
```

## test

```bash
yarn run test
```

## production

Assuming you have installed `pm2`, you can start the production server by script `prod`:

```bash
yarn run prod
```
and freeze your process list by:

```bash
pm2 save
```
