# koa-hello

> clone this as a cli to start your own koa app

## init

```bash
yarn install
```

## run

```bash
yarn run dev
```

## test

```bash
yarn run test
```

## distribute

run the script `build` to get the main file:

```bash
yarn run build
```

then there is a **main.js** in the dir **./dist**. you can run it by script `start`:

```bash
yarn run start
```

or **pm2** as well.
